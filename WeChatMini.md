## 一、平台入驻与认证

1. 全程基于**微信公众平台-小程序**模块操作，小游戏归属小程序类目体系。
2. 完成平台认证，支持**个人认证**或**企业认证**两种类型。
3. 服务类目务必选择**小游戏类目**，类目错误将无法正常提交审核。

## 二、开发账号密钥申请

需在公众平台申请 **AppID（小程序ID）** 和 **AppSecret（小程序密钥）**，二者为微信开发者工具必备参数，用于调试、预览及运行小游戏项目。

## 三、上线前置备案

小游戏提交官方审核**之前，必须完成微信备案**，未完成备案无法进入审核流程。

## 四、版本管理说明

- **开发版本**：代码上传后生成，供内部体验测试使用。
- **审核版本**：提交至微信官方待审核的版本。
- **上线版本**：审核通过后的正式版本，全网用户可搜索并访问。

## 五、成员与体验权限配置

1. 在「成员管理」中添加项目成员、体验成员。
2. 体验人员总计上限 **30 人**，可满足小范围内部测试需求。

## 六、核心网络域名配置

开发管理内的**服务器域名白名单**为关键配置。

小游戏所有数据上传、下载等网络请求，仅放行白名单内域名，需提前将业务所用域名添加至白名单，否则会出现网络访问异常。

需重点配置：

- **request 合法域名**：后端接口域名，例如 `https://api.lyzlearn.com`
- **downloadFile 合法域名**：资源下载域名，如音频、图片、配置文件等
- **uploadFile 合法域名**：如后续涉及用户上传文件，需要单独配置
- **业务 H5 域名**：如客服文本中跳转的玩家中心域名，例如 `https://app.lyzlearn.com`

## 七、客服消息配置

客服消息用于在小游戏内打开微信客服会话，并向用户发送玩家中心、问卷、词库贡献等入口文本。

### 7.1 开启客服能力

1. 进入微信公众平台。
2. 打开「功能」或「客服」相关配置入口。
3. 确认已启用**小程序客服**能力。
4. 如需人工客服接入，可在「客服人员」中绑定客服微信号。

### 7.2 配置消息推送

在「开发管理」中配置**消息推送**：

- **URL（服务器地址）**：填写后端客服消息接收接口，例如 `https://api.lyzlearn.com/weChat/customer-service`
- **Token（令牌）**：填写后端用于微信签名校验的 Token，例如 `WannaEnglishCustomerService`
- **EncodingAESKey（消息加密密钥）**：按微信后台生成值填写
- **消息加密方式**：可选择明文模式或安全模式，需与后端解析方式一致
- **数据格式**：XML

配置完成后，微信会将用户进入客服会话、发送消息等事件推送到该服务器地址。

### 7.3 后端处理要求

后端客服接口需支持：

1. `GET /weChat/customer-service`  
   用于微信服务器 URL 验证，校验通过后返回 `echostr`。

2. `POST /weChat/customer-service`  
   用于接收微信推送的客服事件或用户消息。

3. 校验微信签名  
   使用 `signature`、`timestamp`、`nonce` 和 Token 计算 SHA1，确认请求来自微信。

4. 收到客服进入事件后返回 `success`  
   避免微信判定服务异常。

5. 发送客服文本消息  
   通过微信客服消息接口发送文本。

6. 配置后端环境变量  
   后端需配置微信客服消息相关环境变量，确保验签、获取 access_token、生成玩家中心链接均可正常工作。

推荐配置：

```env
WECHAT_MESSAGE_TOKEN=WannaEnglishCustomerService
H5_HOME_URL=https://app.lyzlearn.com/
```

说明：
- `WECHAT_MESSAGE_TOKEN`：需与微信公众平台「消息推送」中配置的 Token 保持一致。
- `H5_HOME_URL`：客服文本中发送给用户的玩家中心地址。
- 修改环境变量后，需要重新部署或重启后端服务。

### 7.4 客服文本入口规范

当前阶段客服功能只需要发送玩家中心文本入口即可。

推荐文本格式：

```text
WannaEnglish 玩家中心
https://app.lyzlearn.com/?userID=用户ID
```

说明：

- 链接域名必须是已部署、可访问的 H5 域名。
- 用户识别优先使用 `userID`。
- 不建议在链接中附带中文昵称，避免微信中显示较长的 URL 编码。
- 如果页面打不开，优先检查域名解析、HTTPS 证书、微信后台域名配置。

### 7.5 客户端调用要求

小游戏客户端打开客服会话时，应调用微信客服会话能力。

当前目标是发送纯文本入口，因此应关闭自动小程序卡片：

```js
showMessageCard: false
```

说明：

- `showMessageCard: true` 会自动生成微信小程序卡片。
- 如只需要客服返回文字链接，应使用 `showMessageCard: false`。
